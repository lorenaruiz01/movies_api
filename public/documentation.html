MyFlix Movie Database App

Work In Progress

Creating a REST API for an application called “myFlix” that interacts with a database that stores data about different movies. Building the client-side component of this same application using REACT.

My Flix API Documentation

Request URL HTTP Method Request body data format Response body data format
Get a list of ALL movies /movies GET none A JSON object holding an array with all movies.
Get data about a single movie by title /movies/:Title GET None A JSON object holding data about a single movie, containing description, genre, director, image URL, and whether it’s featured or not property. Example: { "Title": "The French Dispatch", "Description": "A love letter to journalists...", "Genre": { "Name": "Comedy", "Description": "Is a genre of fiction that ..." }, "Director": { "Name": "Wes Anderson", "Bio": "Wesley Wales Anderson was born in Houston, Texas...", "Birth": 1969.0 }, "ImageURL": "https://upload.wikimedia.org/wikipedia/en/7/78/The_French_Dispatch.jpeg", "Featured": false }
