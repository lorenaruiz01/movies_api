MyFlix Movie Database App

Work In Progress

Creating a REST API for an application called “myFlix” that interacts with a database that stores data about different movies. Building the client-side component of this same application using REACT.

My Flix API Documentation

Request URL HTTP Method Request body data format Response body data format
Get a list of ALL movies /movies GET none A JSON object holding an array with all movies.
Get data about a single movie by title /movies/:Title GET None A JSON object holding data about a single movie, containing description, genre, director, image URL, and whether it’s featured or not property. Example: { "Title": "The French Dispatch", "Description": "A love letter to journalists...", "Genre": { "Name": "Comedy", "Description": "Is a genre of fiction that ..." }, "Director": { "Name": "Wes Anderson", "Bio": "Wesley Wales Anderson was born in Houston, Texas...", "Birth": 1969.0 }, "ImageURL": "https://upload.wikimedia.org/wikipedia/en/7/78/The_French_Dispatch.jpeg", "Featured": false }
Get data about a genre by genre's name /movies/genre/:genreName GET none A JSON object holding data about a genre containing name and description properties. Example: "Genre": { "Name": "Comedy", "Description": "Is a genre of fiction that ..." }
Get data about a director by director's name /movies/directors/:directorName GET none A JSON object holding data about a director. Example: { "Name": "Wes Anderson", "Bio": "Wesley Wales Anderson is an American filmmaker. His films are known for their eccentricity, unique visual and narrative styles.", "Birth": "1969-05-01T00:00:00.000Z" }
Add a new user /users POST A JSON object holding user data, structured like this: { "Username": "Kim", "Password": "ula36ula", "Email": "kim.namura@fakeema.il", "Birthday": "1984-07-18" } A JSON object holding user data plus an ID and a array for placing favorite movies: { "Username": "Kim", "Password": "ula36ula", "Email": "kim.namura@fakeema.il", "Birthday": "1984-07-18T00:00:00.000Z", "FavoriteMovies": [], "_id": "64de58a81b1e8dfa2e81f5b9", "__v": 0 }
Update user info /users/:Username PUT A JSON object holding the user info that should be changed. It should contain at least one of the following key/value pairs: { "Username": String, "Password": String, "Email": String, "Birthday": Data } A JSON object holding all user info with all key/values pair updated: { "Username": "Kim-Sophie", "Password": "ula36ula", "Email": "kim.namura@fakeema.il", "Birthday": "1984-07-18T00:00:00.000Z", "FavoriteMovies": [], "_id": "64de58a81b1e8dfa2e81f5b9", "__v": 0 }
Add a movie to user favorite list /users/:Username/movies/:movieID POST None A JSON object holding all user info with all key/values pair updated: { "Username": "Kim-Sophie", "Password": "ula36ula", "Email": "kim.namura@fakeema.il", "Birthday": "1984-07-18T00:00:00.000Z", "FavoriteMovies": ["64dcd768591669f639751ff8"], "_id": "64de58a81b1e8dfa2e81f5b9", "__v": 0 }
Remove a movie from user favorite list /users/:Username/movies/:movieID DELETE None A JSON object holding all user info with all key/values pair updated: { "Username": "Kim-Sophie", "Password": "ula36ula", "Email": "kim.namura@fakeema.il", "Birthday": "1984-07-18T00:00:00.000Z", "FavoriteMovies": [], "_id": "64de58a81b1e8dfa2e81f5b9", "__v": 0 }
